# 배열 선언

자바스크립트에서 배열은 변수에 대괄호로 값을 묶어주고 쉼표로 구분만 해주면 된다.
```
var ary = ["아무 값이나", null, 5, '섞여도 상관없다.'];
var arraysInArrays = [[1, 2, 3], ["Julia", "James"], [true, false, true, false]];
```
