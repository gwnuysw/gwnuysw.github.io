# BGP(Border Gateway protocol version 4)
경로벡터 라우팅을 사용하는 인터도메인 라우팅 프로토콜이다.

# 멀티캐스트 라우팀

## 유니캐팅
1:1통신
## 멀티캐스팅
1:n 통신 하나의 패킷이 목적지로 가는 도중 복사되어 멀티캐스트 그룹에 나뉘어 간다.
### 멀티 캐스팅대 다중 유니캐스팅
멀티 유니캐스팅은 N개의 패킷을 N개의 목적지로 맞춰서 전달되기 때문에 멀티 캐스팅이 아니다.

### 멀티 캐스트 응용

* 분산 데이터 베이스 접근
* 정보보급
* 원격회의
* 원격 교육
